## Getting Started

1. npm install to install all the packages of the package.json

2. Add a .env with your data:

   ALCHEMY_ID=
   RPC_URL_INK_SEPOLIA=
   



3. run the development server:

```bash
npm run dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
